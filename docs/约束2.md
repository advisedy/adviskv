```
sequenceDiagram
      participant Meta
      participant SDM
      participant RG as ReplicaGroupReconciler
      participant Leader as Storage Leader
      participant New as New Storage Node

      Meta->>SDM: AlterTable(replica_count=4)
      SDM->>SDM: TablePhase=RESIZING<br/>TableSpec.replica_count=4

      RG->>RG: PlanReconciler<br/>group.target_replica_count=4

      RG->>RG: MembershipReconciler<br/>healthy_count=3 < target=4
      RG->>RG: create new Replica metadata<br/>desired=PRESENT, phase=CREATING<br/>member_type=NON_MEMBER
      RG->>RG: add new rid into desired_members

      New->>SDM: Heartbeat(no new replica)
      SDM-->>New: PRESENT(new replica)<br/>initial_members=desired_members

      New->>New: create local replica
      New->>SDM: Heartbeat(new replica READY, NON_MEMBER)

      Leader->>SDM: Heartbeat(leader)
      SDM-->>Leader: ADD_MEMBER(new replica)

      Leader->>Leader: Raft config change<br/>add new replica as member
      New->>SDM: Heartbeat(new replica READY, VOTER)

      RG->>RG: all desired members READY+VOTER
      SDM->>SDM: TablePhase RESIZING -> READY
```
